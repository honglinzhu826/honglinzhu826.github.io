# 你好，世界

欢迎来到我的个人博客！这是我使用 **Astro** 构建的新博客网站。

## 为什么选择 Astro？

Astro 是一个现代化的静态站点生成器，具有以下优势：

1. **零 JS 默认** - 页面加载极快
2. ** Islands 架构** - 按需加载 JavaScript
3. **内置 i18n** - 原生支持国际化
4. **Content Collections** - 类型安全的 Markdown 管理
