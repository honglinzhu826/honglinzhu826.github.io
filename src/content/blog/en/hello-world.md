# Hello, World

Welcome to my personal blog! This is my new blog website built with **Astro**.

## Why Astro?

Astro is a modern static site generator with these advantages:

1. **Zero JS by default** - Pages load extremely fast
2. **Islands architecture** - Load JavaScript only when needed
3. **Built-in i18n** - Native internationalization support
4. **Content Collections** - Type-safe Markdown management
